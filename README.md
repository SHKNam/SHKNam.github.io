# Seunghyun Nam | AI & Data Science Portfolio

**Building AI systems that integrate automation, data science, LLM based AI Agents, Graph RAG, and Knowledge Graphs into real world business workflows.**

🌐 **Live Portfolio:** https://shknam.github.io

This portfolio showcases my journey from enterprise RPA automation to Data Science, LLM based AI Agents, Knowledge Graphs, Graph RAG, and AI driven decision support systems.

It features **15 end to end projects**, each presented with:

* Problem Definition
* System Architecture
* Implementation
* Results

The portfolio demonstrates how I design and build end to end AI systems, from business automation and predictive analytics to enterprise AI applications powered by LLMs and Knowledge Graphs.

---

## Tech Stack

* **Frontend:** HTML, CSS, Vanilla JavaScript
* **Visualization:** Python generated SVG architecture diagrams
* **Hosting:** GitHub Pages

---

## Repository Structure

```text
index.html                   Main application
css/style.css                Styles and design tokens
js/data.js                   Single source of portfolio content
js/main.js                   Rendering logic and UI interactions
assets/diagrams/             Architecture diagrams (SVG)
assets/pdf/                  Resume, publications, and project reports
tools/generate_diagrams.py   Diagram generation script
```

---

## Maintenance

### Update portfolio content

```text
js/data.js
```

### Regenerate architecture diagrams

```bash
python3 tools/generate_diagrams.py
```

### Run locally

```bash
python3 -m http.server 8000
```
